# **Paxcan Popgen Repository**

This repository contains scripts and files used to analyze the population genomics of *Paxistima canbyi* using GBS data.

---

## **Contents**

### **1. Files and hierfstat Analyses**
This R Markdown file (`.Rmd`) imports the `.vcf` file generated by iPyrad along with the `.strata` file assigning individuals to populations.  
The workflow converts the `.vcf` into **genlight** and **hierfstat** objects, calculates F-statistics and heterozygosity metrics, performs an initial PCA, and runs a Bartlett test of variance on He and Ho.

### **2. adegenet DAPC**
This `.Rmd` uses the **genlight** object to run a DAPC (discriminant analysis of principal components) using the *adegenet* package. It also includes PCA output and a compositional population-assignment plot.

### **3. LEA Analyses**
This `.Rmd` performs sNMF clustering (a STRUCTURE-like approach) using the *LEA* package to infer the number of genetic clusters (K).

### **4. dartR Analyses**
This `.Rmd` uses the *dartR* package to subset the **genlight** object and run population-genetic analyses.  
The workflow includes a Mantel test and an interactive PCA based on *adegenet*’s `glPCA`. *dartR* is also useful for filtering and dataset refinement.
